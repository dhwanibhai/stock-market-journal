Documenting the Journey of Our Stock Market Simulator Project

Project Timeline

  • October 30: Brainstormed and finalized the idea for a stock market simulator. Outlined core functionalities, including real-time trading and portfolio tracking.
  • November 1: Installed essential development tools and libraries. Set up Git repository and project folder structure.
  • November 8: Developed a secure login system. Referenced GitHub repositories for inspiration.
  • November 12: Integrated AlphaVantage API for market data. Faced API key invalidation due to incorrect setup, delaying integration by a day.

  • November 18: Resolved a critical bug in user portfolio updates. Debugging revealed an issue with rounding errors in financial calculations.
  • November 20: Implemented the buy/sell functionality with real-time pricing. Calculated brokerage fees dynamically for each transaction. Encountered issues with delayed API responses causing transaction errors.
  • November 21: Optimized the trading algorithm for better performance. Identified a major bug where incorrect stock prices were being fetched due to time-zone differences.
  • November 23: Developed a funds management system, allowing users to deposit, withdraw, and view transaction history. Encountered data persistence issues during withdrawal operations.
  • November 25: Introduced stock trend graphs with interactive features. Users could compare up to three stocks simultaneously. Faced challenges in rendering graphs on older browser versions.
