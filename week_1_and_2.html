Week 1

Challenges and Solutions

  • Challenge: Couldn’t generate the API Access Token initially.
  • Solution: Resolved by checking the API setup process and correcting the authentication steps.

  • Solution: Immediately revoked the key and updated the .gitignore to prevent future pushes.

Challenges and Solutions

  • Challenge: Path issues and not changing the working directory properly.
  • Solution: Fixed directory structure and worked on file organization.
  • Challenge: Had to filter through 85,000+ instruments to get the correct instrument token for API requests.
